Une application React pour la gestion des produits avec fonctionnalités de mise à jour.


## Installation

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/votre-username/PCZone.git


2. Installez les dépendances :

```bash
cd PCZone
npm install
```

3. Lancez l'application :

```bash
npm start
```
